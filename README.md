REQUIREMENTS
------------

Agar aplikasi berjalan dengan baik, perhatikan kebutuhan aplikasi berikut ini. Pastikan sistem anda sudah memenuhi
syarat-syarat kebutuhan yang ditentukan sebagai berikut:

1. PHP >= 7.4
2. Postgresql >= 10 or MariaDB >= 10.2
3. NodeJS
4. Composer
5. Bower
